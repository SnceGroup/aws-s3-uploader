# aws-s3-utils

AWS Helper for uploading/downloading file to S3 bucket

## Requirements

- [Docker](https://www.docker.com/get-started)

## Installation (for dummies)

- Clone the repository (`git clone git@github.com:SnceGroup/aws-s3-utils.git`).
